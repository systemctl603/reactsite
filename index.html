React JS

A JS library to create a reactive UX\UI.

Features

  • Can be used to quickly create proffessional websites.
  • One codebase can be used for PWA's, websites, and apps.

Philosphy

The React philosophy is composed of components and classes that have a render
function. The render function returns a HTML element or a container that holds
multiple elements, as a render function can return only one HTML element.
The syntax used is called JSX, which is a concatenation of JS and XML.
